# Pothole Hero Mobile App

Built with React Native, Expo, NativeWind, and Supabase.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npx expo start
   ```

## Features

- **Home**: Interactive map showing reported potholes.
- **Report**: Take photos and report potholes with automatic location detection.
- **Dashboard**: View recent reports and their status.

## Configuration

Supabase credentials are configured in `lib/supabase.ts`.
